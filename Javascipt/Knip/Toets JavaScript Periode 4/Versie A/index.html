Toets JavaScript & jQuery

Inleiding

Deze toets test je kennis en vaardigheden op het gebied van JavaScript & jQuery (hoofdstuk 5,6 en 7 uit de reader).

Toetsonderdelen

De toets bevat twee onderdelen. Theorie en praktijk.

Theorie

Het theoretische deel van deze toets bestaat uit 10 meerkeuze-vragen. Alle vragen gaan over jQuery selectoren. Per goed antwoord krijg je 1 punt. Het maximaal aantal te behalen punten voor het theoretische deel is 10 punten.

Praktijk

Het praktische deel van deze toets bestaat uit 3 programmeer opdrachten. Per goed antwoord krijg je een maximum van 10 punten. Het maximaal aantal te behalen punten voor het praktische deel is 30 punten. De programmeeropdrachten maak je in je normale JavaScript / jQuery editor (WEBMATRIX/Notepad++/Sublime/Brackets). Vergeet je code niet te testen! Het toevoegen van commentaar aan je code levert punten op! Kopieer, zodra je script werkt zoals is gevraagd, de gevraagde code naar het antwoord-veld. Verlies je je sessie met N@tschool laat dan je toets door de docent resetten, en start de toets opnieuw. Markeer daarom al je code in je editor, zodat je ze meteen weer opnieuw in de antwoord velden kunt knippen-en-plakken.

Tijd

Voor het theoretische deel krijg je maximaal 20 minuten de tijd. Voor het praktijkdeel krijg je maximaal 45 minuten de tijd.

Succes!
